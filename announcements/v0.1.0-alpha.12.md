# reinhardt-web v0.1.0-alpha.12

## Highlights

This is a maintenance release with no user-facing feature additions or bug fixes. All changes are scoped to developer tooling and internal documentation.

The primary infrastructure addition is a dedicated GitHub Actions workflow (`.github/workflows/copilot-setup-steps.yml`) that provisions `uv` and `uvx` in Copilot's ephemeral runner environment. This allows MCP servers — such as Serena — to be launched via `uvx` during Copilot coding agent sessions, improving the developer experience for contributors using GitHub Copilot on this project.

Alongside that, internal contributor documentation receives a new `docs/GITHUB_INTERACTION.md` file establishing guidelines for how Claude Code interacts with GitHub PRs and issues, covering comment policies, review response formats, and attribution conventions. `CLAUDE.md` is updated with cross-references to these guidelines.

## Breaking Changes

No breaking changes in this release.
