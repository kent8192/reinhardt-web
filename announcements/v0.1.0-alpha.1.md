# reinhardt-web v0.1.0-alpha.1

## Highlights

This is the inaugural alpha release of `reinhardt-web`, the unified facade crate for the Reinhardt full-stack API framework for Rust. Inspired by Django, Django REST Framework, and FastAPI, it exposes a single entry point into an ecosystem of 70+ independent crates covering everything from ORM and REST serialization to GraphQL, WebSockets, dependency injection, and a WASM-based reactive frontend layer. Feature presets (`minimal`, `standard`, `full`, `api-only`, `graphql-server`, `websocket-server`, `cli-tools`, `test-utils`) let users dial in exactly the capabilities and binary size their project needs, while fine-grained feature flags provide further control over authentication backends, database drivers, and middleware.

The release also ships the initial versions of two sub-crates: `reinhardt-shortcuts` (v0.1.0-alpha.2) and `reinhardt-i18n` (v0.1.0-alpha.2), providing keyboard shortcut support and internationalization infrastructure respectively. Both replace yanked v0.1.0-alpha.1 packages that had packaging issues.

Several publishing-readiness fixes land alongside the feature work. Circular dev-dependencies in `reinhardt-macros` that blocked `cargo publish` are resolved by migrating affected integration tests into the top-level `tests/` crate and inlining minimal trait definitions into UI test files. `reinhardt-rest` now bundles its branding assets directly inside the crate rather than referencing workspace-root paths, ensuring `include_bytes!` calls succeed when downloaded from crates.io. A spurious `test-database` feature on `reinhardt-graphql` that pointed at a dev-dependency is removed.

The `reinhardt-micro` crate is removed entirely; its request helper methods (bearer token extraction, client IP resolution, query string parsing) are folded into `reinhardt-http`, consolidating the HTTP surface area. The publish workflow gains Trusted Publishing (OIDC) support via `rust-lang/crates-io-auth-action`, eliminating the need for long-lived API tokens stored in GitHub Secrets.

## Breaking Changes

No breaking changes in this release.
