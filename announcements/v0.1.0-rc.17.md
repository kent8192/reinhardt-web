# reinhardt-web v0.1.0-rc.17

## Highlights

This release focuses on bringing the `reinhardt-commands` project and app templates fully up to date with API changes introduced in rc.13 through rc.15, and fixing long-standing usability issues in the `reinhardt-admin` CLI.

The scaffolding templates across all four project variants (pages, restful, pages-workspace, restful-workspace) now correctly reflect current framework conventions. `base.example.toml` moves to the nested `[core]` TOML format (e.g., `[core.databases.default]`, `[core.security]`), replacing the deprecated flat structure that `settings.rs.tpl` had already moved past. All `models.rs.tpl` files now demonstrate correct `#[user]` macro usage — with the right import path (`reinhardt::macros::user`), required parameters, and properly attributed struct fields including a `created_at` field using the idiomatic `#[field(auto_now_add = true)]` pattern.

Template comments have also been improved to surface important rc.14/rc.15 behaviors: `server_fn.rs.tpl` now notes that `#[inject]` is auto-detected and `use_inject = true` is no longer needed; `index.html.tpl` clarifies that `StaticFilesMiddleware` handles WASM script injection automatically; and `views.rs.tpl` adds a commented example demonstrating proper `JwtError` variant matching for typed JWT authentication error handling.

On the CLI side, `reinhardt-admin startproject` and `reinhardt-admin startapp` replace the broken `--template-type` flag with a required, mutually-exclusive argument group: `--template <TYPE>`, `--with-pages`, or `--with-rest`. Clap now enforces exactly one selection at parse time, surfacing clear errors instead of silently misconfiguring the generated project. The underlying `CommandContext` options are also corrected so the scaffolding commands receive the flags they actually read.

Finally, the `Query` extractor now uses a typed `TokenQuery` struct instead of a `HashMap`, providing stronger type safety for token-based query parameters.

## Breaking Changes

No breaking changes in this release.
