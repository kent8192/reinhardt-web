# reinhardt-web v0.1.0-rc.30

## Highlights

The `#[model]` macro gains a typestate `Model::build()` constructor so adding required fields stays non-breaking for callers. `SessionMiddleware` now auto-registers its `Arc<SessionStore>` via the new `Middleware::di_registrations()` hook, eliminating the manual DI wiring that previously caused `#[inject] session: SessionData` handlers to return a silent 500.

A public SPA navigation API lands: `use_router()`, `navigate()`, and `RouterHandle` dispatch imperative navigation through whichever SPA router is installed, falling back gracefully to `window.location.set_href` when no router is present. The `form!` macro fixes `success_url:` (fires on submit, not on initial mount) and resolves E0434 for closures that capture outer-scope locals in `initial:`, `watch:`, and the new `on_success_ref:` lifted callback.

`#[user]` now auto-emits `BaseUserManager` and `SuperuserCreator` for any `#[user] + #[model]` type without requiring `full = true`. New typed session extractors (`SessionValue<T>`, `SessionValueNamed<K, T>`, and optional variants) accept handler parameters directly without `#[inject]`. `SessionAuthExt::login`/`logout` wraps the four-step session-rotation sequence in a single call, replacing per-handler boilerplate.

Foreign-key column types and nullability now resolve from the target model's registered primary key; `NOT NULL` derives from `FieldState.nullable`, fixing silently nullable columns for non-`Option` fields. SQLite recreation reads column metadata through the open schema editor transaction to prevent schema-lock races. `ALTER COLUMN` rollback dispatches per backend (MySQL: `MODIFY COLUMN`; SQLite: table recreation; CockroachDB: sentinel-row lock replacing the unsupported `pg_advisory_lock`).

`ClientRouter::merge` is now public for multi-app SPA composition. `ViewSet` routes produce typed `ResolvedUrls` accessors and a triple-slash URL composition bug under `mount()` prefixes is fixed. `#[url_patterns]` emits typed `urls::*` helper functions per named route. The new `#[dto]` macro handles `cfg_attr(native, derive(Validate, Schema))` boilerplate for wire-level types. `runserver --with-pages` auto-mounts the project-level `static/` directory.
