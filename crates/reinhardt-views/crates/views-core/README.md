# views-core

Core view abstractions for Reinhardt framework

## Overview

`views-core` provides the fundamental view abstractions and base classes for building views in the Reinhardt framework, including view traits, mixins, and common patterns.

## Features

- Core view trait definitions
- View base classes
- Generic view mixins (list, detail, create, update, delete)
- Request/response handling for views
- View composition patterns
- Common view utilities

## Installation

```toml
[dependencies]
views-core = "0.1.0"
```
