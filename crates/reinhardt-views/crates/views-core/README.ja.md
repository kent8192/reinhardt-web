# views-core

Reinhardtフレームワークのコアビュー抽象化

## 概要

`views-core`は、ビュートレイト、ミックスイン、共通パターンを含む、Reinhardtフレームワークでビューを構築するための基本的なビュー抽象化とベースクラスを提供します。

## 機能

- コアビュートレイト定義
- ビューベースクラス
- 汎用ビューミックスイン（list、detail、create、update、delete）
- ビュー用のリクエスト/レスポンス処理
- ビューコンポジションパターン
- 共通ビューユーティリティ

## インストール

```toml
[dependencies]
views-core = "0.1.0"
```
