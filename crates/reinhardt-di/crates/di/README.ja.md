# di

Reinhardtフレームワークのコア依存性注入システム

## 概要

`di`は、型安全な依存関係解決を備えたFastAPI風の依存性注入パターンを可能にする、Reinhardtフレームワークのコア依存性注入抽象化を提供します。

## 機能

- コア依存性注入トレイト
- 型安全な依存関係解決
- リクエストスコープ依存関係
- シングルトン依存関係サポート
- ファクトリパターンサポート
- 自動依存関係解決

## インストール

```toml
[dependencies]
di = "0.1.0"
```
