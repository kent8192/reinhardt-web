# backends-pool

データベースコネクションプールバックエンド抽象化

## 概要

`backends-pool`は、Reinhardtフレームワークにおけるデータベースコネクションプーリング用のバックエンド抽象化を提供します。依存性注入サポートを備えたデータベースコネクションプールを管理するためのトレイトとユーティリティを定義します。

## 機能

- コネクションプールバックエンド抽象化
- 非同期コネクション管理
- sqlxコネクションプールとの統合
- 依存性注入サポート (オプション)
- スレッドセーフなコネクション処理
- コネクションライフサイクル管理

## インストール

```toml
[dependencies]
backends-pool = "0.1.0"
```

### 機能

- `reinhardt-di`: 依存性注入統合
