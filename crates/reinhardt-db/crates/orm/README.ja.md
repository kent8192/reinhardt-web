# reinhardt-orm

Query

Set APIを備えたDjango風ORM

## 概要DjangoのORMにインスパイアされた強力なオブジェクトリレーショナルマッピングシステムです。チェーン可能なクエリのためのQuerySet API、モデル定義、フィールド型、バリデーター、リレーションシップ管理、複数のデータベースバックエンド（PostgreSQL、MySQL、SQ

Lite）のサポートなどの機能が含まれます。

## 機能

- チェーン可能なクエリ用QuerySet API
- モデルフィールドタイプ（CharField、IntegerFieldなど）
- フィールドバリデータと制約
- データベースリレーションシップ（ForeignKey、ManyToMany、OneToOne）
- モデルマネージャーとカスタムQuerySet
- 集約とアノテーション
- データベーストランザクション
- 複数データベースサポート
