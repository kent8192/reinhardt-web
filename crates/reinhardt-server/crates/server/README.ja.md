# reinhardt-server

HTTPサーバー実装

## 概要

Hyperベースの高性能HTTPサーバーです。リクエストルーティング、ミドルウェアサポート、WebSocket接続、オプションのGraphQLサポートを提供します。async/awaitによる並行接続処理を行います。

## 機能

- 高性能HTTP/1.1及びHTTP/2サーバー
- Tokioによる非同期リクエスト処理
- ミドルウェアパイプラインサポート
- WebSocket接続処理（オプション）
- GraphQL統合（オプション）
- コネクションプーリングとライフサイクル管理
- グレースフルシャットダウンサポート
- リクエストタイムアウト処理
