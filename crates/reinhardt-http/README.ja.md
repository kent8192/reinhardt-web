# reinhardt-http

HTTPリクエスト及びレスポンス処理

## 概要

Reinhardtフレームワークのコア HTTP抽象化を提供します。リクエストとレスポンス型、ヘッダー処理、クッキー管理、JSON、Form、マルチパートデータのサポートを含むコンテントネゴシエーションを提供します。

## 機能

- HTTPリクエスト及びレスポンスの抽象化
- ヘッダーの解析と操作
- クッキー処理（取得、設定、削除）
- コンテントネゴシエーション（Acceptヘッダー）
- JSONリクエスト/レスポンスボディ処理
- フォームデータ解析（application/x-www-form-urlencoded）
- マルチパートフォームデータサポート
- クエリ文字列解析
