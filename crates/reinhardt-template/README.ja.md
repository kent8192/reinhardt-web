# reinhardt-template

Reinhardtフレームワークのテンプレートシステム

## 概要

`reinhardt-template`は、テンプレートエンジン統合、レンダリングユーティリティ、テンプレートマクロを含む、Reinhardtアプリケーション向けの包括的なテンプレートシステムを提供します。

## 機能

このクレートは以下のサブクレートから機能を再エクスポートしています：

- **Templates** (`reinhardt-templates`): コアテンプレートエンジン
  - Askamaベースのテンプレートエンジン（Jinja風構文）
  - `{{ variable }}`構文による変数置換
  - 制御構造（`{% if %}`、`{% for %}`）
  - テンプレート継承（`{% extends %}`、`{% block %}`）
  - データ変換用の組み込みフィルター
  - テンプレートローダーシステム
  - テンプレートキャッシングとセキュリティ機能

- **Templates Macros** (`reinhardt-templates-macros`): テンプレート関連手続き型マクロ
  - コンパイル時テンプレートコンパイル
  - 型安全なテンプレート参照
  - テンプレートID生成

- **Renderers** (`reinhardt-renderers`): レスポンスレンダリングユーティリティ
  - JSONレンダラー
  - HTMLレンダラー
  - テンプレートレンダラー
  - コンテンツタイプネゴシエーション
  - カスタムレンダラーサポート
