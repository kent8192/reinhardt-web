# utils-core

Core utilities for Reinhardt framework

## Overview

`utils-core` provides fundamental utility functions and types used across the Reinhardt framework, including common helpers, type conversions, and core abstractions.

## Features

- Common utility functions
- Type conversion helpers
- String manipulation utilities
- Collection helpers
- Time and date utilities
- Encoding/decoding utilities
- Core abstraction types

## Installation

```toml
[dependencies]
utils-core = "0.1.0"
```
