# utils-core

Reinhardtフレームワークのためのコアユーティリティ

## 概要

`utils-core`は、一般的なヘルパー、型変換、コア抽象化を含む、Reinhardtフレームワーク全体で使用される基本的なユーティリティ関数と型を提供します。

## 機能

- 一般的なユーティリティ関数
- 型変換ヘルパー
- 文字列操作ユーティリティ
- コレクションヘルパー
- 時刻と日付のユーティリティ
- エンコード/デコードユーティリティ
- コア抽象化型

## インストール

```toml
[dependencies]
utils-core = "0.1.0"
```
