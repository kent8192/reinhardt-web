# reinhardt-routers-macros

ReinhardtフレームワークのためのプロシージャルマクロによるURLルーティング

## 概要

`reinhardt-routers-macros`は、ReinhardtフレームワークにおけるURLルーティング定義を簡素化するプロシージャルマクロを提供し、コンパイル時のルート検証と型安全なURLパターン生成を可能にします。

## 機能

- コンパイル時のルート検証
- 型安全なURLパターン生成
- ルート登録マクロ
- 自動ルート名生成
- ViewSetルーティングマクロ
- パスパラメータ検証

## インストール

```toml
[dependencies]
reinhardt-routers-macros = "0.1.0"
```
