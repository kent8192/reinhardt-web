# reinhardt

Reinhardtフレームワークのメインエントリーポイントおよびアンブレラクレート

## 概要

これは、Reinhardtフレームワークのすべてのコアコンポーネントを再エクスポートするメインクレートです。RustでフルスタックREST APIを構築するための統一されたインターフェースを提供し、すべてのサブクレートの機能を単一の便利なパッケージにまとめています。

## 機能

### 実装済み ✓

- **フルスタックAPIフレームワーク** - すべてのコアコンポーネント(ORM、ViewSets、Routers、Authなど)の完全な再エクスポート
- **Django風のアーキテクチャ** - おなじみのパターン: Models、Views、Serializers、Middleware
- **型安全な開発** - コンパイル時保証を備えた完全なRust型システム
- **バッテリー同梱アプローチ** - REST API開発に必要なすべてが1つのクレートに
- **複数の構成フレーバー** - Micro、Standard、Full構成用のフィーチャーフラグ
  - Micro: ルーティング + パラメータ + DIのみ
  - Standard (デフォルト): コアREST API機能
  - Full: 管理画面、GraphQL、WebSocketsを含むすべての機能
- **Preludeモジュール** - 一般的なインポート用の便利な再エクスポート
- **フィーチャーベースのモジュール性** - データベース、キャッシュ、セッションなどのオプション機能

### 予定

- 特定のユースケース向けの追加のフィーチャーフラグの組み合わせ
- より細かいフィーチャー制御
