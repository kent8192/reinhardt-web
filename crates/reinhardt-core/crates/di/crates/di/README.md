# di

Core dependency injection system for Reinhardt framework

## Overview

`di` provides the core dependency injection abstractions for the Reinhardt framework, enabling FastAPI-style dependency injection patterns with type-safe dependency resolution.

## Features

- Core dependency injection traits
- Type-safe dependency resolution
- Request-scoped dependencies
- Singleton dependency support
- Factory pattern support
- Automatic dependency resolution

## Installation

```toml
[dependencies]
di = "0.1.0-alpha.1"
```
