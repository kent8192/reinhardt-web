# reinhardt-backends

Reinhardtフレームワークの共有バックエンドインフラストラクチャ

## 概要

このクレートは、スロットリング、キャッシング、セッションストレージを含む、Reinhardtフレームワークのさまざまなコンポーネント間でデータを保存および取得するための統一されたバックエンドシステムを提供します。

## 機能

- **Backendトレイト**: TTLサポートを備えた汎用キーバリューインターフェース
- **MemoryBackend**: 自動有効期限付きの高性能インメモリストレージ
- **RedisBackend**: Redisを使用した分散ストレージ（フィーチャーゲート）
- 非同期ファーストデザイン（`async-trait`使用）
- TTLサポートによる自動有効期限
- `serde`による型安全なシリアライゼーション/デシリアライゼーション
- スレッドセーフな並行アクセス
- カウンター用のインクリメント操作
