# reinhardt-browsable-api

HTML形式のブラウザブルAPIインターフェース

## 概要

APIエンドポイントを探索およびテストするためのWebベースのインターフェースです。API応答の人間に優しいHTMLレンダリング、エンドポイントをテストするためのインタラクティブなフォーム、および認証処理を提供します。

## 機能

### 実装済み ✓

#### コアレンダリング

- **BrowsableApiRenderer**: HandlebarsベースのHTMLテンプレートレンダラー
  - グラデーションヘッダーデザインを採用したデフォルトのDRF風テンプレート
  - カスタマイズ可能なテンプレート登録サポート
  - JSON応答の整形表示とシンタックスハイライト
  - モダンなCSSスタイリングによるレスポンシブデザイン
- **ApiContext**: API レンダリング用の完全なコンテキスト構造
  - タイトル、説明、エンドポイント、HTTPメソッドの表示
  - ステータスコード付きのレスポンスデータ
  - 許可されたHTTPメソッドの可視化
  - テーブル形式でのリクエストヘッダー表示
  - オプションのフォームコンテキスト統合

#### レスポンス処理

- **BrowsableResponse**: 構造化されたAPIレスポンスタイプ
  - serde_json::Valueサポート付きのデータペイロード
  - ステータス、メソッド、パス、ヘッダーを含むResponseMetadata
  - 便利なコンストラクタ (new, success)
  - 完全なシリアライズ/デシリアライズサポート

#### フォーム生成

- **FormContext**: インタラクティブなリクエストフォームのレンダリング
  - 動的なフォームフィールド生成
  - 複数の入力タイプのサポート (text, textarea など)
  - 必須フィールドのインジケーター
  - フィールドガイダンスのためのヘルプテキスト
  - フォームフィールドの初期値サポート
- **FormField**: 個別のフォームフィールド設定
  - フィールド名、ラベル、タイプの指定
  - 必須/オプションフィールドの処理
  - ヘルプテキストと初期値のサポート

#### テンプレートシステム

- **ApiTemplate**: 基本的なHTMLテンプレートユーティリティ
  - シンプルなAPIレスポンスのレンダリング
  - ステータスコード付きのエラーページ生成
  - シンプルなユースケース向けのフォールバックテンプレート

#### ビジュアル機能

- HTTPメソッドバッジの色分け (GET, POST, PUT, PATCH, DELETE)
- 等幅フォントでのエンドポイント表示
- JSON応答用のダークテーマコードブロック
- シャドウエフェクト付きのレスポンシブコンテナレイアウト
- 適切な入力コントロールを備えたフォームスタイリング
- クリーンなフォーマットでのヘッダーテーブル表示
