# rest-core

ReinhardtフレームワークのREST API機能用コア抽象化

## 概要

`rest-core`は、コア型、シリアライザートレイト、ビュー抽象化を含む、ReinhardtフレームワークでREST APIを構築するための基本的な抽象化とトレイトを提供します。

## 機能

- コアREST APIトレイトと抽象化
- シリアライザートレイト定義
- ビューベースクラスとミックスイン
- リクエスト/レスポンス処理抽象化
- コンテンツネゴシエーションサポート
- REST操作用の汎用型定義

## インストール

```toml
[dependencies]
rest-core = "0.1.0"
```
