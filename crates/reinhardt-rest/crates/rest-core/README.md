# rest-core

Core abstractions for REST API functionality in Reinhardt framework

## Overview

`rest-core` provides the fundamental abstractions and traits for building REST APIs in the Reinhardt framework, including core types, serializer traits, and view abstractions.

## Features

- Core REST API traits and abstractions
- Serializer trait definitions
- View base classes and mixins
- Request/response handling abstractions
- Content negotiation support
- Generic type definitions for REST operations

## Installation

```toml
[dependencies]
rest-core = "0.1.0"
```
