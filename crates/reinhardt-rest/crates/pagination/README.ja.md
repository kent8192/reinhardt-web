# reinhardt-pagination

ページネーション戦略 (PageNumber、LimitOffset、Cursor)

## 概要

大規模データセット用の複数のページネーション戦略を提供します。従来のページベースページネーション用のPageNumberPagination、SQLスタイルのlimit/offset用のLimitOffsetPagination、オフセットのパフォーマンス問題なしで大規模データセットを効率的にページ分割するCursor

Paginationが含まれます。

## 機能
- PageNumberPagination
- LimitOffsetPagination
- CursorPagination
- カスタムページネーションクラス
- ページサイズ設定
