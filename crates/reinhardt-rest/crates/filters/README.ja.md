# reinhardt-filters

クエリフィルタリングとソート機能

## 概要

クエリセット用の強力なフィルタリングとソートシステムです。フィールドルックアップ（exact、contains、gt、ltなど）、複数フィールドでの検索、フィールドによるソート、カスタムフィルターバックエンドをサポートします。ORMとView

Setとシームレスに統合されます。

## 機能

- フィールド全体のテキスト検索用SearchFilter
- 結果ソート用OrderingFilter
- カスタムフィルターバックエンド
- フィルタークエリ文字列解析
- フィールド固有のフィルタリング
