# reinhardt-serializers

型安全なデータシリアライゼーションとバリデーション

## 概要

Rust型とJSON/その他の形式間の変換を行うシリアライザーを提供し、組み込みのバリデーション機能を備えています。フィールドレベルとオブジェクトレベルのバリデーター、ネストされたシリアライザー、自動モデルシリアライゼーションのためのORMとの統合が含まれます。

## 機能

- データ変換用シリアライザー
- データベースモデル用ModelSerializer
- フィールドレベル検証
- ネストされたシリアライザー
- カスタムフィールドタイプ
- 読み取り専用および書き込み専用フィールド
