# Reinhardt 基礎チュートリアル

実際の投票アプリケーションを構築しながら、Reinhardtフレームワークの基礎を学びます。

## 概要

このチュートリアルシリーズでは、ゼロから完全に機能する投票アプリケーションを作成します。モデル、ビュー、テンプレート、フォーム、テスト、管理画面など、Reinhardtの中核となる概念を学習します。

## 対象読者

このチュートリアルは以下の方々を対象としています：

- Reinhardtを初めて学ぶ開発者
- DjangoからRustとReinhardtに移行する開発者
- サーバーレンダリングテンプレートを使用したWebアプリケーションを構築する方

## 前提知識

このチュートリアルを始める前に、以下の知識が必要です：

- Rustプログラミングの基礎
- Cargo（Rustのパッケージマネージャー）の使い方
- HTTPの概念とWeb開発の理解
- コードエディタまたはIDE

## 作成するもの

このチュートリアルを通じて、以下の機能を持つ投票アプリケーションを構築します：

ユーザーとして：

- 投票とその選択肢を表示
- 投票を行う
- 投票結果を確認

管理者として：

- 管理画面から投票を作成・管理
- 各投票の選択肢を追加・編集
- 管理画面をカスタマイズ

## RESTチュートリアルとの比較

REST APIの構築にも興味がある場合は、[RESTチュートリアル](../rest/quickstart.md)も確認してください。主な違いは以下の通りです：

- **基礎チュートリアル**: サーバーレンダリングテンプレートを使用したフルスタックWebアプリケーションに焦点
- **RESTチュートリアル**: JSONレスポンスとシリアライザを使用したAPI構築に焦点

どちらのチュートリアルもReinhardtの基本概念を教えるため、一方を学ぶことでもう一方の理解に役立ちます。
