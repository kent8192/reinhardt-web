# パート7: 管理画面のカスタマイズ

このチュートリアルでは、Reinhardt管理画面を探索し、投票データを管理するためにカスタマイズする方法を学びます。

## 管理画面の有効化

Reinhardt管理画面は、アプリケーションのデータを管理するための強力で自動生成されるインターフェースです。投票アプリケーション用に有効化しましょう。

まだ追加していない場合は、管理画面の依存関係を追加します：

```toml
[dependencies]
reinhardt = { version = "0.1.0", features = ["admin"] }
```

## 管理者ユーザーの作成

まず、管理画面にアクセスできるスーパーユーザーアカウントを作成します：

```rust
// src/main.rsに追加するか、別の管理セットアップスクリプトを作成
use reinhardt::prelude::*;

async fn create_superuser(pool: &SqlitePool) -> Result<(), Box<dyn std::error::Error>> {
    let user = User::create_superuser(
        pool,
        "admin",
        "admin@example.com",
        "password123",
    ).await?;

    println!("Superuser created: {}", user.username);
    Ok(())
}
```

これを一度実行して管理者ユーザーを作成します。
